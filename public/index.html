Kord AI Registration API

API Overview

This API allows you to register bots and retrieve their information. Below are the endpoints available for use.

Endpoints

1. Register a Bot

POST /api/register-bot

This endpoint is used to register a new bot.

Request Body:

{
  "ownerName": "John Doe",
  "ownerNumber": "1234567890",
  "botName": "ChatBot",
  "botType": "Chatbot"
}

Response:

{
  "message": "Bot registered successfully"
}

2. Retrieve a Bot

GET /api/register-bot?Name=John%20Doe&Number=1234567890

This endpoint retrieves information about a registered bot.

Query Parameters:

  • Name: The name of the bot owner.
  • Number: The phone number of the bot owner.

Response:

{
  "message": "Bot found",
  "bot": {
    "ownerName": "John Doe",
    "ownerNumber": "1234567890",
    "botName": "ChatBot",
    "botType": "Chatbot"
  }
}
Note: Ensure that all required fields are filled out when making requests.

Usage Example

You can use tools like Postman or curl to interact with the API.

Example Using curl:

curl -X POST http://localhost:3000/api/register-bot \
-H "Content-Type: application/json" \
-d '{"ownerName": "John Doe", "ownerNumber": "1234567890", "botName": "ChatBot", "botType": "Chatbot"}'
