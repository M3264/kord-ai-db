Kord AI Registration API

This API allows you to register bots and retrieve their information.

Register a Bot

POST /api/register-bot

curl -X POST /api/register-bot \
-H "Content-Type: application/json" \
-d '{"ownerName": "JohnDoe", "ownerNumber": "1234567890", "botName": "ChatBot", "botType": "Chatbot"}'

Parameters:

  • The name of the bot owner.
  • The contact number of the bot owner.
  • The name of the bot.
  • The type of the bot.

Register a Bot (GET)

GET /api/register-bot

curl "/api/register-bot?ownerName=JohnDoe&ownerNumber=1234567890&botName=ChatBot&botType=Chatbot"

Parameters:

  • The name of the bot owner.
  • The contact number of the bot owner.
  • The name of the bot.
  • The type of the bot.

Retrieve Bot Information

GET /api/get-bot

curl "/api/get-bot?Name=JohnDoe&Number=1234567890"

Parameters:

  • The name of the bot owner.
  • The contact number of the bot owner.
